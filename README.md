# WorthIt Privacy Policy

Static privacy policy site for the WorthIt Android app.

## Files

* `docs/index.html` redirects visitors to the privacy policy.
* `docs/privacy-policy.html` contains the policy page.

## Before Publishing

Replace the placeholders in `docs/privacy-policy.html`:

* `[YOUR NAME OR BUSINESS NAME]`
* `[YOUR CONTACT EMAIL]`
* `[DATE]`

## GitHub Pages

To publish this repository with GitHub Pages:

1. Open the repository on GitHub.
2. Go to **Settings**.
3. Open **Pages**.
4. Set **Source** to **Deploy from a branch**.
5. Set **Branch** to **main**.
6. Set the folder to **/docs**.
7. Select **Save**.

After GitHub Pages finishes deploying, the privacy policy will be available at the repository's
GitHub Pages URL.
